 - [MongoDB](https://mongodb.com/)
 - [JWT](https://jwt.io/)

# Project setup
 1. Clone this git repository
 2. Install dependencies
    ```bash
    $ npm install
    ```
 3. Compile and Run project
	 ```bash
	 # development
	 $  npm  run  start
	 
	 # watch mode
	 $  npm  run  start:dev
	 
	 # production mode
	 $  npm  run  start:prod
	 ```
